Automate IP warmup for Postal mail servers with advanced templating and analytics.

== Description ==

Postal Warmup Pro is the ultimate solution for managing Postal mail server warmup directly from WordPress.

**Key Features:**

*   **Multi-Server**: Manage multiple Postal instances.
*   **Secure**: AES-256 encryption for credentials.
*   **Performance**: Optimized for high volume with Action Scheduler.
*   **Analytics**: Detailed insights into delivery performance.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/postal-warmup` directory.
2. Run `composer install` to load dependencies.
3. Activate the plugin through the 'Plugins' screen in WordPress.
4. Configure your servers in the Postal Warmup menu.
